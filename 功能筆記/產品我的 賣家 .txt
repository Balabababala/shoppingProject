API 路徑	HTTP 方法	說明	備註
/api/seller/products	GET	取得賣家所有商品清單	需登入賣家

/api/seller/products/{id}	PUT	編輯指定商品	根據 ID 修改商品資料
/api/seller/products/{id}	DELETE	刪除指定商品	根據 ID 刪除商品
/api/seller/products/{id}	GET	取得指定商品詳細資料	編輯用
